Description List of HTML Tags

p
The p tag tag represents a paragraph of text.
a
The a tag tag creates hyperlinks, allowing you to link to other web pages or resources.
img
The img tag is used to embed images in a web page.
ul
The ul tag tag defines an unordered list, typically represented with bullet points.
h2
The h2 tag tag defines a level 2 heading, which is typically used for subsections of content.
